# Avishka Sanduni Portfolio Website

## 📌 Project Title
Personal Portfolio Website – Avishka Sanduni  
Information Systems Student & Japanese Language Tutor

---

## 💻 Technologies Used
- HTML5
- CSS3
- Bootstrap 5.3
- Bootstrap Icons
- JavaScript (Bootstrap bundle)
- Google Fonts (system fonts used)

---

## ✨ Features Implemented

### 🧭 Layout & Structure
- Fully responsive layout using Bootstrap Grid System
- Section-based design (Hero, About, Skills, Contact)
- Clean and modern UI structure

### 📱 Responsiveness
- Mobile-first design approach
- Responsive navbar with hamburger menu
- Adaptive columns using `col-md-*`
- Flexible stacking using `flex-column-reverse`

### 🎨 UI Components Used
- Navbar (sticky-top, collapse)
- Cards for skills section
- Buttons with Bootstrap styling
- Badges for highlight text
- Icons using Bootstrap Icons
- Rounded profile image

### 🧩 Design Features
- Dark navy theme for branding
- Monospace font style for tech feel
- Smooth spacing using Bootstrap utilities
- Shadow and border utilities for modern look
